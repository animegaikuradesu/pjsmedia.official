Mencari Tuhan & Menghancurkan Berhala

Sejak muda, Ibrahim kritis mencari Tuhan. Ia mengamati bintang, bulan, dan matahari, lalu menyadari semua itu terbenam dan bukan Tuhan. Ia pun berdakwah kepada ayahnya (pembuat patung) dan Raja Namrud. Suatu hari, Ibrahim menghancurkan berhala-berhala kaumnya dan menyisakan yang paling besar untuk menyadarkan logika mereka.

Dibakar Api & Membangun Ka'bah

Karena perbuatannya, Namrud membakarnya hidup-hidup dalam api raksasa. Namun Allah memerintahkan api menjadi dingin: "Hai api menjadi dinginlah, dan menjadi keselamatanlah bagi Ibrahim". Di masa tua, bersama anaknya Ismail, beliau membangun Ka'bah di Mekkah dan menyerukan ibadah Haji.
